Use Case Scenario: Remove Art from Portfolio

Context: The user is logged in and on his or her portfolio

Primary Flow:
1. The user finds the art to remove from the portfolio
2. The user clicks on the art
3. The user clicks on the "Remove" button
4. The user confirms the removal of the art

Alternative Flows:
The user cancels the remove operation
A4. The user clicks the "Cancel" button
A5. The user is taken to his or her profile

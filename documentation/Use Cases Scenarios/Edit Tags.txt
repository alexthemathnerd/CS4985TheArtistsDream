Use Case Scenario: Edit tag

Context: User is logged in

Primary Flow:
1. User navigates to their portfolio page
2. User click on the manage portfolio button
3. User clicks the Art post that they would like to edit
4. User enters additional tags or removes current tags
5. User clicks the confirmation button
6. System updates tags for art post
7. User is brought back to their profile page

Alternative Flows:
User wants to cancel edits
A5. User clicks cancel button
A6. User is prompted to confirm the cancellation
Return to primary flow step 7.

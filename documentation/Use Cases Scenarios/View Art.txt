Use Case Scenario - View Art


Primary Flow:
1. The user logs in to the system
2. The user sees the first page, the recommended art page

Alternative Flows:
The user views art from artists followed
A1. The user navigates to the page for artists followed
A2. The user views the art from artists followed

The user selects to load more art into view
A2. The user has seen the maximum amount of art allowed on the screen
A3. The user selects the button "View More" to load more art
A4. The user views the additionally loaded art

The user views a single artwork
A2. The user clicks on a single piece of artwork
A3. The user sees the artwork in a larger view - a popup window

Use Case Senario: Add art

Context: The user is logged in and on his or her profile page.

Primary Flow:
1. The user clicks on the "Manage Profile" button
2. The user selects the "add art to portfolio" option
3. The user is able to select artwork from the file explorer to be able to add art to the portfolio
4. The user confirms that the selected art is the desired art to add to portfolio

Alternative Flows:
The user selects the remove art option
A2. The user selects the "remove art from portfolio" option
A3. See remove art scenario

The user selects the edit tags option
A2. The user selects the "edit tags" option
A3. See edit tags scenario

The user cancels the add operation
A4. The user clicks cancel
A5. The user is brought back to the profile page
